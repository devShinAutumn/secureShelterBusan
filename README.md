🛡️ SecureShelterBusan

부산 지역의 대피소 기반 재난 대응 정보를 한눈에 볼 수 있는 대시보드 서비스입니다.

SecureShelterBusan은
	•	부산 전역의 대피소 위치 정보,
	•	각 대피소의 현재 수용 가능 인원(잔여 자리),
	•	재난 상황 발생 시 즉각 확인해야 할 핵심 정보,
을 직관적으로 제공하여 시민들의 신속한 의사결정을 돕는 것을 목표로 합니다.

🔍 주요 기능
	•	대피소 현황 조회: 실시간 잔여 자리 및 수용 가능 인원 표시
	•	지도 기반 탐색: 가까운 대피소를 위치 기반으로 빠르게 확인
	•	재난 정보 대시보드: 기상·지진·재난 관련 핵심 지표 시각화
	•	안전 안내 알림: 상황 변화에 따른 실시간 업데이트(옵션)

🎯 목적

재난 발생 시 “어디로 가야 하는지, 지금 자리가 있는지”를
누구나 쉽게 확인하고 빠르게 행동할 수 있도록 돕는 서비스입니다.
